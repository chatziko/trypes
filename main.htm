Καλωσήρθατε στη σελίδα των Τρυπών στο World Wide Web. Στη σελίδα αυτή μπορείτε να βρείτε πολλές πληροφορίες για το συγκρότημα, για τους δίσκους του, για νέα ή συναυλίες αλλά και στίχους, εικόνες και ηχητικά αποσπάσματα απο τραγούδια. Υπάρχει ακόμα και μια On-Line δημοσκόπηση στην οποία μπορείτε να λάβετε μέρος. Επιπλέον πληροφορίες, εικόνες η οτιδήποτε υλικό είναι κάτι παραπάνω από ευπρόσδεκτα. Για συμβουλές, πληροφορίες ή κριτική μην διστάσετε να επικοινωνήσετε μαζί μου μέσω της σελίδας σχολίων. Για την καλύτερη εμφάνιση της σελίδας αυτής χρησιμοποιήστε Netscape Navigator καθώς στη σελίδα χρησιμοποιούνται Java Applets.
