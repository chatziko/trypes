Οδηγίες Για Την Χρήση Ελληνικών Στις Φόρμες Του Netscape Navigator ver. 2.0

Η χρήση ελληνικών στις φόρμες του Netscape 2.0 είναι εφικτή αλλά για να μπορέσει να γίνει θα πρέπει από πριν να έχουμε κάνει κάποιες απλές αλλά απαραίτητες ρυθμίσεις. Οταν στο Netscape 2.0 ρυθμίζουμε τις γραμματοσειρές (Options / General Preferences / Fonts) ορίζουμε μια γραμματοσειρά μεταβλητού πλάτους (Proportional Font) και μία σταθερού πλάτους (Fixed Font) την οποία το Netscape 2.0 χρησιμοποιεί στις φόρμες. Ομως, για λόγους που ακόμα δεν έχω καταλάβει, ακόμα και αν αυτή η γραμματοσειρά υποστηρίζει τα ελληνικά, στις φόρμες δεν εμφανίζονται. Αντίθετα εμφανίζονται όταν ορίσουμε την ίδια γραμματοσειρά και σαν μεταβλητού αλλά και σαν σταθερού πλάτους. Αρα αυτό που πρέπει να κάνετε είναι το εξής : Επιλέξετε διαδοχικά Options / General Preferences / Fonts και στο πλαίσιο "For the encoding" επιλέξτε "Latin2". Ορίστε τώρα σαν Proportional και Fixed μια γραμματοσειρά σταθερού πλάτους που να υποστηρίζει ελληνικά (πχ. Courrier Greek ή Courrier και New Greek). Αν δεν διαθέτετε μια τέτοια γραμματοσειρά κάντε κλικ ΕΔΩ για κα κατεβάσετε το αρχείο "fonts.zip" που περιέχει μεταξύ άλλων και τη γραμματοσειρά "Courrier New Greek" που κάνει για την περίπτωση. Τέλος πατήστε "OK" και τελιώσατε. Τώρα κάθε φορά που θέλετε να συμπληρώσετε μια φόρμα θα επιλέγετε διαδοχικά Options / Document Encoding / Central European (Latin2). Επειδή βέβαια η σελίδες θα εμφανίζονται δυσανάγνωστες με αυτό το encoding, μόλις τελιώσετε με τη φόρμα επιλέξτε διαδοχικά Options / Document Encoding / Western (Latin1).
