On-Line Δημοσκόπηση
Στη σελίδα αυτή οργανώνεται μια On-Line δημοσκόπηση με θέμα τη δισκογραφία των Τρυπών. Στη δημοσκόπηση αυτή μπορείτε να λάβετε μέρος ακολουθώντας τις οδηγίες που υπάρχουν παρακάτω. Επειδή όμως η σελίδα αυτή είναι ακόμα καινούρια, τα αποτελέσματα θα αρχίσουν να δημοσιεύονται σε λίγο καιρό, μόλις ο αριθμός των χρηστών που ελάβαν μέρος μεγαλώσει αρκετά, ώστε να μπορεί να βγει ένα αποτέλεσμα που να εκφράζει την πλειοψηφία των οπαδών και όχι μεμονωμένων ατόμων.


    Οδηγίες Συμμετοχής
    Η συμμετοχή στη δημοσκόπηση είναι μια ιδιαίτερα απλή διαδικασία. Το μόνο που έχετε να κάνετε είναι να απαντήσετε στις 4 ερωτήσεις που δίνονται παρακάτω μέσω της σελίδας συμμετοχής. Στην περίπτωση που ο browser σας δεν υποστηρίζει φόρμες στείλτε τις απαντήσεις μέσω ενός e-mail το οποίο θα πρέπει να έχει ως τίτλo "TRYPES - Dhmoskophsh" και επίσης θα πρέπει να αναφέρετε και το όνομά σας, το οποίο θα καταχωρηθεί στη λίστα των χρηστών που έλαβαν μέρος. Για να συμμετάσχετε στη δημοσκόπηση θα πρέπει να απαντήσετε και στις 4 ερωτήσεις έτσι ώστε να έχουμε μια ευρύτερη ιδέα για τις απόψεις σας. Για επιπλέον πληροφορίες επικοινωνήστε μαζί μου στην γνωστή διεύθυνση.


    Ερωτήσεις
    Οι παρακάτω ερωτήσεις είναι οι 4 ερωτήσεις της δημοσκόπησης. Πρέπει να σημειωθεί ότι εάν ένα τραγούδι έχει βγει σε Live ή ακουστική έκδοση, αυτό πρέπει να σημειωθεί στην απάντηση αλλιώς το τραγούδι θα μετρήσει στην κανονική του έκδοση.
